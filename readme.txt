SpeedPress for WooCommerce is a collection of lightweight, performance-focused WooCommerce addons designed to improve speed, usability, and conversion rates for your online store.

== Description ==

SpeedPress for WooCommerce brings a set of modular addons to enhance your WooCommerce store without adding bloat. 

Key features:

* Auto-apply coupons based on cart conditions
* Free shipping notices in cart and checkout
* Product view tracking
* WooCommerce speed optimization
* Cart and checkout performance improvements
* Easily extendable modular design
* Compatible with most WooCommerce themes

Each addon is designed to be lightweight and improve user experience while keeping your store fast.

== Installation ==

1. Upload the `speedpress-for-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the individual addons from the SpeedPress settings page (if available).

== Frequently Asked Questions ==

= Is this plugin compatible with all WooCommerce themes? =

Yes, SpeedPress for WooCommerce is designed to work with most themes that follow WooCommerce standards.

= Can I enable/disable individual addons? =

Yes, each addon is modular and can be toggled individually from the plugin settings page.

= Will this plugin affect my store speed? =

No, SpeedPress is performance-focused. It is designed to **improve speed** by optimizing WooCommerce features and removing unnecessary scripts.
